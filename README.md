# QuickCalculator-67
A simple and fast web calculator built with HTML, CSS, and JavaScript.
add UI framework info to README

## 功能特点
- 基本的四则运算（加、减、乘、除）
- 清空功能
- 百分比计算
- 响应式设计，支持多种设备屏幕

## 如何使用
1.  点击仓库中的 `index.html` 文件
2.  在文件详情页右键，选择 "Open with Live Server" (如果你安装了VSCode插件) 或直接下载后用浏览器打开
3.  在网页上点击按钮即可进行计算

## 技术栈
- **前端**: HTML5, CSS3, JavaScript (ES6)
- **版本控制**: Git & GitHub
- **UI框架**: 无，使用纯CSS实现
